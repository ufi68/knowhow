## Install 
Download last version from [Download jq](https://stedolan.github.io/jq/download/) 
for example (centos)
``` 
wget https://github.com/stedolan/jq/releases/download/jq-1.6/jq-linux64 
chmod x+a jq
mv jp /usr/local/sbin
```
